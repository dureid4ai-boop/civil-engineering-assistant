# SAMPLE — Rebar Weight Estimate (Synthetic)

Inputs:
- d = 16 mm
- L_total = 120 m
- Waste/laps = 3%

Unit weight:
w = d² / 162 = 16² / 162 = 256 / 162 = 1.580 kg/m (approx)

Gross weight:
W_gross = 1.580 × 120 = 189.6 kg

Multiplier:
M = 1 + 3/100 = 1.03

Net weight:
W_net = 189.6 × 1.03 = 195.3 kg

Result:
Estimated rebar weight ≈ 195 kg
